## ğŸ”„ How the System Works

1. Phone numbers and appointment data are stored in Google Sheets  
2. Vapi initiates inbound or outbound calls using Twilio  
3. Riley listens to the user and detects intent  
4. Structured outputs are generated by the assistant  
5. Make.com updates appointment status in Google Sheets  
6. Call results can be viewed in Vapi call logs
